# No Fandom
Removes fandom.com links from popular search engines to allow for indie wikis to populate the search results in their stead.

## Installation
Installation requires the uBlock Origin addon/extension for [Firefox](https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/) or [Chrome](https://chromewebstore.google.com/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm) based browsers.
### One-click Link
Once you have uBlock Origin installed, you can subscribe to this blocklist by clicking [this link](https://subscribe.adblockplus.org/?location=https://raw.githubusercontent.com/themathics/no-fandom/refs/heads/main/no-fandom.txt&title=No%20Fandom).
### Manual Installation
Alternatively, you can install it manually by
1. Enter uBlock Origin settings (click on the extension at the top of thr browser, then click the cogwheel button).
2. Select "Filter Lists" (found at the top of the settings options for uBlock Origin).
3. Scroll all the way to the bottom, at the bottom of the page select "Import"
4. Copy and paste the URL `https://raw.githubusercontent.com/themathics/no-fandom/refs/heads/main/no-fandom.txt` into the box.
5. Select "Apply Changes"

## Currently Supported Search Engines
- google.com
- startpage.com
- duckduckgo.com
- bing.com
- search.brave.com
